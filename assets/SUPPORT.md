# Support

## Public Support Scope

Public support for ProofChain AI Enterprise is limited to:

- public repository navigation
- public documentation questions
- public landing page issues
- broken public links
- clarification of the Public Edition boundary
- responsible disclosure routing

Public support does not include private source review, implementation support, deployment guidance, acquisition negotiation, legal review, or transfer package access.

## Private Acquisition and Licensing

Private acquisition, licensing, transfer, or partnership discussions must be handled outside public GitHub threads.

Schedule a private discussion:

https://cal.com/ciprian-stefan-plesca

## Security Support

For security concerns, do not open a public issue.

Follow [SECURITY.md](SECURITY.md).

## Response Expectations

This repository does not provide guaranteed public support response times.

Any paid support, transfer support, post-acquisition assistance, diligence support, or implementation support must be defined in a separate written agreement.

## Public Communication Rules

Do not post any of the following in public issues, discussions, or pull requests:

- secrets, tokens, keys, or credentials
- private source logic
- internal prompts
- customer or buyer data
- confidential acquisition terms
- wallet material
- production infrastructure details
- third-party confidential information
