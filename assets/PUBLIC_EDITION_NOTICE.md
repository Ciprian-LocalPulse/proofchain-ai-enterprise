# Public Edition Notice

## Purpose

This repository is the Public Edition of ProofChain AI Enterprise.

It exists to provide a procurement-safe, acquisition-ready, public-facing view of the product concept, brand, governance posture, and evaluation surface.

## Public Edition Includes

The Public Edition may include:

- sanitized product overview
- public landing page
- preview assets
- non-sensitive executive documentation
- ownership and license notices
- support and security policies
- contribution rules
- public funding configuration

## Public Edition Excludes

The Public Edition does not include:

- private source logic
- prompts or AI orchestration logic
- deployment secrets
- API keys, tokens, credentials, or wallet material
- internal business strategy
- customer data or buyer data
- proprietary implementation details
- private technical handover documents
- production infrastructure configuration
- private smart contract deployment material
- confidential acquisition terms

## Private Enterprise Transfer Package

The Private Enterprise Transfer Package is separate from this public repository.

Private package access, private source review, transfer rights, exclusivity, commercial deployment rights, and support obligations are available only through explicit written agreement.

The public repository should not be treated as the complete product, a complete source package, or a production deployment package.

## Public Discussion Boundary

Public issues and pull requests must remain limited to the public documentation and public preview surface.

Do not request or disclose private source logic, prompts, credentials, customer data, internal strategy, private architecture, or restricted implementation details in public GitHub channels.

## Procurement-Safe Interpretation

This repository may support initial review, but it is not:

- a security audit
- a compliance certification
- a legal opinion
- a production readiness attestation
- a transfer agreement
- an open source release

Formal diligence, security review, legal review, or acquisition review must occur privately under the appropriate written terms.
