# Public Edition Notice

ProofChain AI Enterprise is published here as a Public Edition for enterprise evaluation, public documentation, portfolio review, trust/provenance discussion, and commercial discovery.

## Purpose of the Public Edition

The Public Edition is intended to communicate:

- lawful provenance and verification product direction
- architecture-level thinking
- public governance posture
- interface and documentation quality
- procurement-safe positioning
- commercialization readiness without exposing private implementation details

## What the Public Edition May Include

The public repository may include:

- README and product positioning
- static landing page or public interface preview
- sanitized architecture descriptions
- non-operational examples
- governance documents
- safe support and security reporting instructions
- high-level commercial evaluation language

## What the Public Edition Does Not Include

The Public Edition does not include and should not be used to infer:

- private smart contract implementation details
- production deployment methods
- wallet, signing, or key-management operations
- Supabase schemas, storage rules, policies, or service-role flows
- Stripe configuration, payment flows, webhooks, or secrets
- certificate-generation internals or anti-tamper logic
- API keys, private keys, seed phrases, tokens, credentials, or environment variables
- customer, buyer, or client data
- private playbooks, prompts, runbooks, or restricted research
- commercial implementation details
- acquisition terms beyond intentionally published public materials

## Private Enterprise Edition

The Private Enterprise Edition, if made available, is separate from this public repository and may only be accessed under a written agreement.

Private Enterprise Edition discussions should remain private and should not be opened as public GitHub issues when they involve implementation detail, pricing, buyer requirements, contract deployment, wallet operations, payment architecture, customer environment assumptions, or confidential evaluation criteria.

## Responsible Use

ProofChain AI Enterprise is positioned for lawful trust, provenance, verification, legal-tech, compliance, and enterprise evaluation purposes. Public materials must not be used to support fraud, false provenance claims, credential misuse, unauthorized access, payment abuse, contract abuse, or unsafe operational activity.

## Public Communication Standard

Issues, pull requests, discussions, and references to this repository should remain calm, factual, enterprise-safe, and security-conscious. Do not publish sensitive findings or private implementation assumptions in public channels.
